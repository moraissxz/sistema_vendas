# 🍦 Sistema de Vendas para Sorveteria

Este projeto consiste no desenvolvimento de um sistema web completo para gerenciamento de vendas em uma sorveteria, com foco na experiência do usuário, organização de pedidos e integração prática com canais de atendimento.

A aplicação permite que clientes naveguem por um cardápio digital, adicionem produtos ao carrinho e finalizem pedidos de forma simples e intuitiva, com integração direta ao WhatsApp para comunicação rápida. Além disso, o sistema conta com autenticação de usuários e painel administrativo para gerenciamento eficiente dos dados.

O backend foi desenvolvido utilizando Python com Django, garantindo uma estrutura robusta, segura e escalável. No frontend, foram utilizadas tecnologias como HTML, CSS, JavaScript e Bootstrap, proporcionando uma interface moderna, responsiva e otimizada para diferentes dispositivos.

Entre as principais funcionalidades, destacam-se o carrinho de compras dinâmico, sistema de login e registro de usuários, listagem de produtos, controle de pedidos e exibição de mensagens de feedback. O sistema também utiliza o Django Admin para facilitar o gerenciamento de produtos e usuários.

A arquitetura do projeto segue boas práticas de desenvolvimento web, com separação clara entre frontend, backend e arquivos estáticos, além de organização modular que facilita manutenção e evolução do sistema.

Atualmente, o sistema utiliza SQLite como banco de dados padrão, podendo ser facilmente adaptado para PostgreSQL em ambientes de produção.

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de engenharia de software, desenvolvimento web full stack e boas práticas de codificação, sendo uma base sólida para expansão futura com funcionalidades como integração de pagamentos, controle de estoque, dashboards administrativos e deploy em nuvem.
